🛠️ Windows 11 AtlasOS Kurulum Rehberi

AtlasOS ile daha hızlı, temiz ve özelleştirilmiş bir Windows deneyimi yaşayın.

AtlasOS Nedir?

AtlasOS, Windows 11'i optimize eden, gereksiz bloatware'ları (gereksiz yazılımları) kaldıran, performansı artıran ve gizliliği artıran bir özelleştirilmiş Windows kurulumudur. Açık kaynak topluluk tarafından geliştirilir ve kişisel kullanım için uygundur.

Neden AtlasOS?

  • ⚡ Daha hızlı başlangıç ve yanıt süresi
  • 🚫 Microsoft bloatware'larından arındırılmış
  • 🔒 Gizlilik odaklı ayarlar
  • 🎨 Modern ve temiz arayüz
  • 💾 Daha az disk kullanımı

📦 İndirme ve Kurulum

  1. AtlasOS ISO Dosyasını İndirin:
    GitHub - AtlasOS/Windows
  2. USB Hazırlayın:
    Rufus veya Ventoy kullanarak bir USB belleği önyükleme yapılabilir hâle getirin.
  3. USB ile Başlatın:
    Bilgisayarınızı USB'den başlatın (BIOS/UEFI ayarlarından).
  4. Kurulumu Başlatın:
    Windows kurulum sihirbazını takip edin. Bölümleme yaparken dikkatli olun.
  5. Tamamlayın ve Güncelleyin:
    Kurulum sonrası Windows Update ile sürücüleri ve güvenlik yamalarını yükleyin.

⚠️ Uyarılar

AtlasOS üçüncü parti bir dağıtım olup, Microsoft tarafından desteklenmez.
- Yedek almayı unutmayın.
- Lisans gereksinimleri hâlâ geçerlidir (orijinal Windows lisansı önerilir).
- Test ortamında denemeniz önerilir.

❓ Sık Sorulan Sorular

Q: AtlasOS güvenli mi?
A: Kaynağı güvenilir GitHub reposundan indirildiğinde genellikle güvenlidir, ancak her zaman dosya hash'lerini kontrol edin.

Q: Güncellemeler ne olur?
A: Windows Update çoğu güncellemeyi getirir, ancak bazı özelleştirmeler geçici olarak bozulabilir.
